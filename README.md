# 🍕 Pizza Sales Dashboard Analysis

## 📌 Project Overview

This project analyzes pizza sales data to uncover key business insights such as top-selling products, customer preferences, and sales trends. The dashboard is built to help stakeholders make data-driven decisions.

## 🛠️ Tools Used

* SQL (Data Extraction & Analysis)
* Excel / Power BI (Dashboard Creation)

## 📊 Dashboard Highlights

* Total Revenue: $817,860.05
* Total Orders: 21,350
* Total Quantity Sold: 49,574
* Average Orders per Transaction: 2.32

## 📈 Key Insights

* The **Classic category** generates the highest number of orders, especially in the **Large size**, indicating strong customer preference.

* **Large size pizzas** contribute the highest share of sales (46%), making it the most preferred size among customers.

* **Hawaiian pizza** is the top-selling product with the highest number of orders.

* **Brie Carre pizza** has the lowest demand, with orders below 50, indicating poor performance.

* Orders are **highest on weekends (Friday & Saturday evenings)**, showing peak customer activity during this time.

* The busiest hours are **12 PM–1 PM and 5 PM–8 PM**, highlighting lunch and dinner rush periods.

## 📉 Performance Analysis

### 🔝 Top 5 Pizzas

* The Hawaiian Pizza
* The California Chicken Pizza
* The Thai Chicken Pizza
* The Classic Deluxe Pizza
* The Pepperoni Pizza

### 🔻 Bottom 5 Pizzas

* The Brie Carre Pizza
* The Green Garden Pizza
* The Calabrese Pizza
* The Soppressata Pizza
* The Spinach Supreme Pizza

## 🎯 Conclusion

The analysis reveals strong customer preference for Classic and Large-sized pizzas. Additionally, a few underperforming products can be reviewed or removed from the menu, while peak hours and days can be leveraged for promotions and better resource planning.
